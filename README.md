# ☕ Trilha Java Básico

Este repositório reúne todos os projetos, desafios e exercícios desenvolvidos durante a trilha de **Java Básico** da DIO.me.

## 📚 Sobre

Aqui estão organizadas as práticas realizadas ao longo dos estudos, com foco no aprendizado dos principais fundamentos da linguagem Java, incluindo:

* Estruturas de controle (if, else, switch)
* Laços de repetição (for, while)
* Entrada e saída de dados
* Tratamento de exceções
* Lógica de programação
* Simulações simples (banco, validações, etc.)

## 🧠 Objetivo

O objetivo deste repositório é:

* Consolidar o aprendizado em Java
* Praticar lógica de programação
* Criar uma base sólida para estudos mais avançados (Spring, APIs, etc.)
* Servir como portfólio de projetos iniciais

## 📂 Estrutura

Os projetos estão organizados em arquivos separados, cada um representando um desafio ou exercício específico da trilha.

## 🚀 Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Git e GitHub

## 📌 Observações

* Todos os projetos foram desenvolvidos com foco educacional
* Os códigos podem evoluir conforme o aprendizado avança
